# Web3-Labs-From-Ethereum-Basics-to-ZK-Stack
A hands-on code repository exploring smart contract development. This project documents my journey from understanding EOA-contract interactions on Ethereum to implementing scalable transactions on Layer 2 using the ZK Stack.
This repository contains my smart contract implementations and experiments, following the security and development patterns from the Cyfrin/Patrick Collins curriculum. It covers the full lifecycle of a DApp, from local development to Layer 2 scaling.

## 🚀 What's Inside

* **Foundry Development:** Standardized project structure using `forge` for compiling and testing.
* **Transactions & State:** Practical implementations of EOA-to-Contract interactions and internal calls.
* **Layer 2 Operations:** Testing contracts on ZK Stack (zkSync Era) to understand EraVM execution and state diffs.
* **Security & Tests:** Unit, Integration, and Forked-mainnet testing to ensure contract reliability.

## 📁 Repository Structure

* `contracts/`: Solidity source files.
* `script/`: Deployment and interaction scripts (Foundry-based).
* `test/`: Robust test suites (Unit & Fuzzing).
* `lib/`: External dependencies (OpenZeppelin, Forge-std).

## 🛠 Tech Stack
* **Solidity** (Advanced patterns & Gas optimization)
* **Foundry** (Forge, Cast, Anvil)
* **ZK Stack** (L2 Scaling & Account Abstraction)
